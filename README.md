# Sitana

Sitana Framework is multiple platform Games (and Apps) framework based on MonoGame/XNA.

## Supported Platforms

* Windows (XNA)
* iOS
* Android (soon)
* Windows Phone 8 (soon)
* Windows (DirectX) (soon)

## Quick Start

Run Download.exe to download all necessary packages.
